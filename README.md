# LEI-OHM
O Projeto Lei de Ohm é uma aplicação web educacional desenvolvida com foco em aprendizado interativo de conceitos fundamentais da eletricidade, utilizando desenvolvimento front-end e lógica de programação.

O sistema foi criado para demonstrar, de forma visual e dinâmica, a relação entre:

tensão elétrica,
corrente elétrica,
resistência.

Diferenciais do Projeto

O projeto foi desenvolvido buscando unir aprendizado técnico e experiência visual moderna, simulando aplicações reais utilizadas em plataformas educacionais digitais.

Além do foco acadêmico, o sistema também serviu como prática de desenvolvimento web voltado para aplicações funcionais e interativas.
